# 💎 Jewellery Shop Website

A modern and elegant Jewellery Shop Website built using Django. The website provides a beautiful user interface for exploring jewellery collections, shopping online, learning about the brand, and viewing collaborations.

---

## ✨ Features

- 🏠 Attractive Home Page
- 💍 New Collection Section
- 🛒 Shop Online Page
- 🤝 Collaboration Page
- 👩 About Us Page
- 💰 Pricing Section
- 📞 Contact Section
- 🔐 User Authentication (Login & Signup)
- 📱 Responsive Design
- 🎨 Modern UI with Custom CSS

---

## 🛠️ Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite3

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/ashmitaverma209-web/Jewellery-Shop.git
```

Move to project folder

```bash
cd Jewellery-Shop
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the server

```bash
python manage.py runserver
```

Open in browser

```
http://127.0.0.1:8000/
```
